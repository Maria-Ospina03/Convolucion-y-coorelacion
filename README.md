### Descripción
Esta práctica de laboratorio se centra en la aplicación de la convolución para determinar la respuesta de sistemas discretos, la correlación para medir la similitud entre señales y la Transformada de Fourier para el análisis espectral en el dominio de la frecuencia. 

### Introducción
La convolución es una operación matemática que combina dos funciones para describir su superposición. Consiste en desplazar una de las funciones sobre la otra, multiplicar sus valores en los puntos donde se superponen y sumar estos productos para generar una nueva función que representa cómo interactúan ambas entre sí.
